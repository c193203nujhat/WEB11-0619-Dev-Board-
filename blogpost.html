Question-1: What are the different ways to select an element in the DOM?

1. By ID → document.getElementById("myId") (Single element)
2. By Class → document.getElementsByClassName("myClass") (HTMLCollection)
3.By Tag → document.getElementsByTagName("p") (HTMLCollection)
4. By Query Selector → document.querySelector(".myClass") (First match)
5. By Query Selector All → document.querySelectorAll("div") (NodeList)
6.By Name Attribute → document.getElementsByName("username") (NodeList)
7.By document.body and document.head

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Gets or sets the full HTML within an element.

innerText: Gets or sets the rendered text content, as shown on the page.

textContent: Gets or sets the raw text content, exactly as in the HTML source.

Question-3: What is event delegation in the DOM?

JavaScript's event delegation technique involves attaching a single event listener to a parent element so that it can handle events that happen on its child elements, either now or in the future. The event bubbling method is used to record events at the parent level and then identify which child element caused the event, as opposed to assigning individual listeners to each child.

Question-4: What is event bubbling in the DOM?


Event bubbling is the process by which an event on one element sets off event listeners on that element, its parent, its grandparent, and so on up the DOM tree.


Question-5: How do you create, add, and remove elements using JavaScript?


Create: document.createElement('elementName')
Add: parent.appendChild(child) or parent.insertBefore(child, existingChild)
Remove: parent.removeChild(child) or child.remove()
